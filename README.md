## Font Tester

A browser-based tool for comparing two fonts side-by-side.

**Features:**
- Drag & drop font files (TTF, OTF, WOFF, WOFF2)
- Variable font support with axis sliders and named instances
- OpenType feature toggles (ligatures, stylistic sets, etc.)
- Multiple size ranges from 8pt to 72pt
- Mixed preview mode (alternating fonts per word)
- Dark/light theme

**Live:** [typedev.github.io/fonttester](https://typedev.github.io/fonttester/)
